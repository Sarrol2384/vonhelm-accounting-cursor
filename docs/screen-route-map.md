## UX improvements over Sage

- Left sidebar navigation (replaces 12-item top menu with deep flyouts)
- Command palette (Ctrl+K) for universal search
- KPI cards on dashboard
- Compliance traffic-light strip on Company Console
- Non-blocking notification pattern in header
- Empty states with CTAs for notes/tasks
- Status badges on all data tables
